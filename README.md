# ValuesBot

Telegram-бот для прохождения теста на ранжирование личных ценностей по методологии Рокича. Пользователь сравнивает пары и тройки ценностей и в конце получает свой персональный рейтинг важности.

## Как это работает

В классическом тесте Рокича испытуемый получает список из ~18 ценностей и должен полностью отсортировать его по важности — задача, требующая держать в голове весь список и постоянно его перетасовывать. Это утомительно и в большинстве случаев избыточно: на практике людям важно **отличить главное от второстепенного**, а не выстроить идеальный порядок внутри второстепенного.

Этот бот решает ту же задачу иначе: серией коротких бинарных и тернарных выборов вида «что для вас важнее — A или B?» и «какая из этих трёх ценностей наименее важна?». Алгоритм адаптивно строит из ответов структурированный результат.

Тест проходит в **двух категориях** — терминальные (цели жизни) и инструментальные (средства достижения). Внутри каждой:

1. **Сравнение тем.** Ценности заранее сгруппированы в 3–4 тематические группы (например, «Личное развитие», «Отношения и общество», «Статус и безопасность»). На этом шаге бот ранжирует темы попарно — выясняет, какие два направления для вас важнее остальных.
2. **Сравнение ценностей внутри топовых тем.** Ценности из двух самых важных тем сводятся в один пул и полностью сортируются попарно — это и будет ваш точный рейтинг главного.
3. **Тройной отсев в нижних темах.** Для ценностей из менее важных тем полная сортировка не нужна — вместо неё бот показывает их тройками и просит вычеркнуть наименее важную из каждой. Это позволяет отделить «всё-таки важные» от «менее важных» небольшим числом вопросов.

Вопросы из обеих категорий **чередуются**, чтобы тест не превращался в долгий моноблок: после вопроса про терминальные ценности идёт вопрос про инструментальные. Но в одном вопросе всегда сравниваются элементы одного типа.

Под капотом — пошаговая **сортировка слиянием** (merge sort даёт минимальное число сравнений в худшем случае) на этапах 1–2 и **тройное вычёркивание** на этапе 3. Полный прогон укладывается примерно в **100 вопросов** в худшем случае.

**Результат** — три блока в каждой категории:
- **Важные** — упорядоченный список из двух самых важных тем
- **Также важные** — пережившие тройной отсев, без точного порядка
- **Менее важные** — вычеркнутые на этапе тройного отсева

Прогресс сохраняется в SQLite после каждого ответа, можно выйти и вернуться. Команды: `/start` (начать или продолжить), `/undo` (отменить последний выбор), `/restart` (начать заново), `/result` (показать сохранённый результат).

## Запуск

Понадобится [uv](https://docs.astral.sh/uv/) — он сам подтянет нужную версию Python (см. `.python-version`).

Получите токен бота у [@BotFather](https://t.me/BotFather), затем:

```bash
git clone <repository-url>
cd ValuesBot
cp example.env .env             # и впишите BOT_TOKEN
uv sync
uv run python main.py
```

## Разработка

```bash
uv sync --group dev

uv run pytest                       # тесты
uv run pytest --cov=bot             # с покрытием

uv run ruff check bot tests         # линтер
uv run ruff format bot tests        # форматирование
uv run mypy bot                     # типы

uv run pre-commit install           # один раз — хук на коммит
```

Тесты используют компактный каталог `tests/fixtures/values.json`, а не продакшен-данные из `res/values.json`. CI прогоняет ruff, ruff-format, mypy и pytest на Python 3.11 и 3.12.

Все пользовательские тексты собраны в одном месте — `bot/views/texts.py` — это и точка перевода, и единый источник для меню BotFather.

**Known limitation:** защита от гонок при быстрых тапах сделана через `asyncio.Lock` на пользователя в рамках одного процесса polling. Multi-process деплой потребует версионирования строк в БД или внешней блокировки.

## Формат каталога ценностей

`res/values.json` — две категории (`terminal`, `instrumental`), каждая с тематическими группами и словарём ценностей:

```json
{
  "terminal": {
    "name": "Терминальные ценности",
    "groups": [
      {
        "name": "Личное развитие и самореализация",
        "values": {
          "Самосовершенствование": "Постоянный личностный рост..."
        }
      }
    ]
  }
}
```
